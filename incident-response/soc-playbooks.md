## Phishing Playbook
1. Analyze email headers
2. Check URLs & attachments
3. Block sender & domains
4. Search for similar emails
5. Reset compromised credentials

## Malware Detection Playbook
1. Validate hash & sandbox file
2. Identify execution path
3. Isolate endpoint
4. Kill malicious processes
5. Hunt for lateral movement

---

## Suspicious IP Connection
1. Check reputation & ASN
2. Identify affected hosts
3. Correlate logs (Firewall / EDR / SIEM)
4. Block IP
5. Document incident
